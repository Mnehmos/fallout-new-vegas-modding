# FNV Base Game Perk Database
# Source: FalloutNV.esm + community documentation
# Status: Pre-overhaul (vanilla ratings)

## Quick Stats
- Total base game perks: ~88 player-acquirable perks
- Target: All perks reach A or B tier
- D/F tier count (to buff): ~12-15 perks

---

## Tier Ratings Legend
- **S** — Build-defining, game-changing
- **A** — Always worth taking on qualifying build
- **B** — Situationally strong, good on right build
- **C** — Marginal, filler, minor benefit
- **D** — Trap pick, mechanically weak
- **F** — Nearly useless

## Priority Buff List (D/F → Target B)

These need the most design work:

1. **Here and Now** — Completely rework the effect
2. **Explorer** — Add exploration XP bonus
3. **Swift Learner** — Add permanent INT gain
4. **Terrifying Presence** — Add real combat effect
5. **In Shining Armor** — Buff DT and add faction damage bonus
6. **Lead Belly** — Add HP heal on irradiated consumption
7. **Rad Absorption / Rad Resistance** — Buff rates + add SPECIAL bonus
8. **Broad Daylight** — Add daytime PER bonus
9. **Home on the Range** — Add crafting bonus
10. **Walker of the Mojave** — Add location damage bonus

---

## Implementation Order

Phase 1 — Easy wins (simple value changes in FNVEdit):
- Requirements adjustments (level req, SPECIAL req)
- Percentage increases (dmg bonuses, regen rates)

Phase 2 — Effect additions (GECK scripting required):
- Secondary effects on existing perks
- Conditional bonuses

Phase 3 — Full reworks (GECK + NVSE scripting):
- Here and Now replacement
- Terrifying Presence combat effect
- Dynamic bonuses (location-based, time-based)
